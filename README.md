# Online Laptop Store

Това е курсов проект за създаване на динамичен уеб магазин с Client-Server архитектура. 

## Технологии
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)

## Как да стартирате проекта локално:

1. Клонирайте хранилището и влезте в папката:
```
git clone https://github.com/martistanchev/online-laptop-store
cd online-laptop-store
```
2. Инсталирайте нужните зависимости (Express и Mongoose):
```
npm install
```
3. Инициализирайте базата данни:
```
node seed.js
```
4. Стартирайте сървъра:
```
node server.js
```
5. Отворете проекта в браузъра:
* Магазин за клиенти: http://localhost:3000
* Админ панел: http://localhost:3000/admin.html
